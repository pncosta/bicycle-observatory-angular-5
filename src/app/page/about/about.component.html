Sobre

O que é?

O Olhos nas Rodas é uma plataforma criada para incentivar a partilha de informação relativa a bicicletas roubadas.

O seu objetivo é criar uma base de dados única de fácil consulta que permita a qualquer pessoa reportar um roubo ou verificar se uma dada bicicleta foi dada como roubada, ajudando assim não só um dono a poder reencontrar a sua bicicleta roubada, mas também ajudando a reduzir o mercado negro de bicicletas roubadas e aumentando a percepção sobre o problema do roubo de bicicletas, permitindo consultar em que zonas há maior número de roubos, quais os tipos de cadeados mais roubados, etc.

A informação disponível é contrubuida pelos vários utilizadores e é disponibilizada de forma gratuita.

Como colaborar

Existem diversas formas de colaborar com o Olhos Nas Rodas.
A simples ação de reportar uma bicleta roubada e ajudar a construir uma base de dados por si só é a maior ajuda que existe para o projeto.

O projeto é desenvolvido em código aberto, sendo possivel contribuir para o mesmo em github.com.

Sugestões de novas funcionalidades, feedback de erros ou qualquer outro tipo de feedback são bem vindos através de qualquer um dos meios de contacto

Por fim, o Olhos Nas Rodas não foi criado com fins lucrativos mas, como qualquer site, tem custos de manutenção e muitas horas de trabalho investidas.
